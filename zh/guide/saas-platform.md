# TOPSALE SaaS 平台

TOPSALE 在开源 SDK 之外，还提供完整的标签打印 SaaS 解决方案。无论你想通过代码集成还是无需开发直接使用，我们都能满足。

## 选择你的方式

| | SDK 集成 | SaaS 平台 |
|---|---|---|
| **适合** | 构建自定义应用的开发者 | 需要开箱即用方案的企业 |
| **方式** | 将 `@appzgatenz/label-print-topbridge-js` 集成到你的 Web 应用 | 直接使用我们的托管平台 |
| **工作量** | 需要开发 | 零代码，即开即用 |

## 平台资源

### [TOPSALE 官网](https://topsale.biz/solution/label-printing/)

了解 TOPSALE 标签打印完整解决方案，包括功能介绍、定价方案和使用场景。

### [TOPSALE 自助服务](https://service.topsale.co.nz/self-service)

管理账户、配置打印机和模板，获取技术支持资源。

### [TopBridge 下载](https://service.topsale.co.nz/self-service/download/topbridge)

下载最新版本的 TopBridge 桌面应用（支持 Windows 和 macOS）。

### [TOPSALE 购物商店](https://service.topsale.co.nz/store)

浏览和购买标签打印方案、打印机配件和订阅套餐。

### [标签设计器](https://service.topsale.co.nz/self-service/label-template)

使用所见即所得编辑器设计和编辑标签模板 —— 无需安装。

### [Excel 批量打印](https://service.topsale.co.nz/self-service/excel-print)

导入电子表格，通过自定义字段映射批量打印标签。

## 快速开始

1. [下载 TopBridge](https://service.topsale.co.nz/self-service/download/topbridge) 并安装到本地电脑
2. 在 [TOPSALE 自助服务](https://service.topsale.co.nz/self-service) 注册账户
3. 在 [标签设计器](https://service.topsale.co.nz/self-service/label-template) 中设计标签，或使用 [Excel 批量打印](https://service.topsale.co.nz/self-service/excel-print) 批量打印

:::tip 需要更多控制？
如果你想构建自定义的打印体验，[TopBridge JS SDK](/zh/guide/getting-started) 提供对打印工作流的完全编程控制。
:::
